# SpinRank

Milestone 0 foundation for the SpinRank web app.

## Local setup

1. Copy `.env.example` to `.env.local`.
2. Set `VITE_API_BASE_URL` to the Apps Script web app URL for your environment.
3. Install dependencies with `npm install`.
4. Run `npm run dev`.

## Included in milestone 0

- Vite + TypeScript frontend scaffold.
- Typed API contract and DTO definitions for planned actions.
- Environment configuration for dev/prod backend URLs.
- Apps Script `health` action stub.
- GitHub Actions workflow for frontend deployment and backend validation.
